# Glossary

## Song

The top-level validated object used throughout the app.

## Lyric Section

A song section such as `Verse 1` or `Verse 3`.

## Lyric Segment

One line-level lyric unit within a section.

## Melody Package

A grouped lyric chunk within one lyric segment, with its own ordered note list.

## Melody Text

The shorthand YAML authoring format for melody packages, for example
`So = C,B,G`.

## Seed-Managed Song

A song row that still tracks its disk seed and can be auto-refreshed.

## User-Managed Song

A song row that has been manually edited in the app and is no longer
auto-overwritten by seed refresh.
