## Structural Read

This version is compact and guitar-friendly. It presents a simpler chart:

- intro centered around `G Am C D`
- opening verse starts with `G Am`, then resolves through `C D G`
- second half of verse uses `C D C D` on the longer trailing phrase
- pre-refrain/bridge area uses `Am C D`, then `G D`
- refrain uses the main emotional movement `C D G Bm Em`
- final refrain turn uses `Am D C G`
- second verse is abbreviated in the page with the assumption that the earlier pattern repeats

## Chord Ideas Worth Keeping

- `G Am C D` is the main intro/groove vocabulary.
- `C D G Bm Em` is the strong refrain color in this chart.
- `Am D C G` appears as the closing tag for the cafe/cigarette line.
- Several comments challenge whether `Bm` should actually be `B` in the refrain. This is unresolved.
- Several comments suggest `Em D` or `Am Em D` around the "old things / bad habits" area.

- `George` proposes a practical beginner strumming pattern based on down/down/up/up-down/down/down-up, with section-specific chord-change handling.

- `Haris Chtapodeis` writes a correction that lines up strongly with Version 2: the "old things" phrase moves through `Am G D`, the refrain uses `C D G B7 Em`, the inner refrain uses `Am D G B7 Em`, and the cafe/cigarette tag resolves to `Am D G`.
- `CHARIS` argues that there is no `Bm`, but `B`, especially in the refrain. This is an important dispute for the final version.

- `pantelis florina` says a phrase ending on "old things" should use `Em` instead of `C`.
- `k_t` gives an older correction that also points away from plain `C` on the "old things" phrase: use `Am Em D`. The same comment says the `Bm` around the refrain may be unnecessary and that `B7` is better for the inner refrain phrase.
- Several users provide solo/lick fragments, but the comments disagree and should be treated as optional instrumental material, not the core lesson version.

## Rhythm Hypothesis

The strongest shared lesson idea is not a fixed printed strum, but a feel:

- even folk-rock ballad pulse
- accented downstroke at the top of the phrase
- down/down/up/up-down/down/down-up as a practical starting point
- allow split patterns when the chord changes inside the sung line
- keep the intro lighter and let the open strings ring

## Reliability Notes

This version is useful as a baseline because it is older, highly discussed, and explicitly tied to the album version. It is also noisy because the comments include many contradictory corrections. The final SingNote version should probably use this as the primary source for structure, but cross-check chord substitutions against Version 2.
